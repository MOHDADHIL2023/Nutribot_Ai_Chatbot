# 🥗 NutriBot: AI-Powered Nutrition Assistant
NutriBot is an intelligent conversational agent designed to provide personalized nutritional advice, meal suggestions, and healthy lifestyle guidance.

---

## 🌟 Overview
NutriBot leverages Natural Language Processing (NLP) to understand user queries regarding diet and health. It serves as a 24/7 virtual nutritionist, helping users make informed decisions about their eating habits through an interactive chat interface.

## 🛠️ Tech Stack
- **NLP Engine:** Google Dialogflow (ES)
- **Frontend:** HTML5, CSS3, JavaScript
- **Framework:** Bootstrap 5 (for responsive UI)
- **Integration:** Dialogflow Messenger API

## ✨ Key Features
- **Intent Recognition:** Understands varied user inputs from "What should I eat for protein?" to "Suggest a healthy breakfast."
- **Real-time Interaction:** Instant responses via a lightweight web-integrated chat widget.
- **Responsive Design:** Optimized for both desktop and mobile viewing.
- **Custom UI:** Tailored chat interface matching the NutriBot brand aesthetic.

## 🚀 How It Works
1. **User Input:** The user types a health-related question in the chat interface.
2. **Processing:** The query is sent to the Dialogflow agent, where the NLP engine identifies the **Intent** and extracts **Entities**.
3. **Response:** The agent matches the query to a pre-configured response or fulfillment and sends it back to the UI.
